# Alkor SVG Editor

Um editor SVG desenvolvido para estudar a API SVG, viewport, transformações e interações de canvas.

## Objetivos

- Aprender SVG
- Implementar Grid
- Implementar Rulers
- Implementar Pan
- Implementar Zoom
- Implementar Seleção
- Implementar Resize

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS

## Status

MVP Finalizado
Este projeto foi encerrado após atingir seus objetivos de aprendizado.

Ele permanece como referência para estudos de SVG e poderá servir de base para futuros experimentos.
